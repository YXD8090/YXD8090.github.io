Vue 基础

Vue 介绍

Vue 是一套用于构建用户界面的渐进式框架
特点：易用，灵活，高效
兼容性：IE8 以上

使用 Vue

1. 引入 Vue.js 文件
2. 创建 节点 标签
3. 创建 Vue 实例对象(el：挂载节点的选择器, data：所有模板中使用的数据 )
4. 使用 Vue 实例对象中的数据({{变量名/表达式}})

MVVM

M：model ( data 里面的数据)
V：view  (视图，就是模板)
VM：(虚拟模型)

Vue 指令

Vue 指令本质上就是 自定义属性

常用指令

  • v-html     渲染内容，可以渲染 HTML 结构，有安全问题
  • v-text      渲染文本内容
  • v-cloak    防止闪烁(将元素设置 display:none 渲染完成后失效)
  • v-pre       跳过编译
  • v-noce    一次性绑定
  • v-on        事件绑定
  • v-bind     属性绑定(用来修改，添加标签属性)
  • v-model  双向绑定(一般给输入表单或者文本域，视图和数据同步改变)
  • v-if
    • v-if
    • v-else-fi
    • v-else
  • v-for

Vue 事件

绑定

< 标签 v-on : 事件类型 = ‘ 时间处理函数 ‘ / >

传参

< 标签 v-on : 事件类型 = ‘ 事件处理函数( 实参 ) ‘ / >
methods : { 事件处理函数 ( 形参 ) {} }

事件对象

当事件处理函数有小括号时使用 $event 获取事件对象

<标签 v-on : 事件类型 = ‘事件处理函数( $event )’/>
methods : { 事件处理函数 ( 形参 ) {} }

当事件处理函数没有小括号调用时，函数的形参就是事件对象

<标签 v-on : 事件类型 = ‘事件处理函数’/>
methods : { 事件处理函数 ( 形参 ) {} }

传参的同时获取事件对象

<标签 v-on : 事件类型 = ‘事件处理函数( 实参1,$event )’/>
methods : { 事件处理函数 ( 形参1,形参2 ) {} }

修饰符

事件对象修饰符

<标签 @事件类型 . 修饰符 = ‘ 事件处理函数 ‘/>

修饰符内容

  • . stop         阻止事件冒泡
  • . prevent    阻止默认事件
  • . self           非内部元素触发
  • . capture    事件捕获
  • . once        事件只触发一次

按键修饰符

<标签 @事件类型 . 修饰符 = ‘ 事件处理函数 ‘/>

按键相关事件

  • keydown
  • keyup
  • keypress
  • input

自定义按键修饰符

Vue.config.keyCodes.修饰符名 = 按键码

Vue 样式

class 操作样式

1. 对象

<标签 :class=”{ class名1:变量1，class名2:变量2 }” />

注意：变量的值是布尔值，表示是否使用该类名

2. 数组

<标签 :class=”[变量1，变量2,’class名1’,’class名2’]” />

注意：变量的值就是类名，加引号表示类名而不是变量

3. 属性绑定

<标签 :class=”条件 ？ ‘class名1’:’class名2’ “ />

直接添加类名或者表达式

style 操作样式

1. 对象

<标签 :style=”{ css属性名1:css属性值1，css属性名2:css属性值2 }” />

2. 数组

<标签 :style=”[变量1，变量2]” />

注意：数组中的变量都是对象，对象中的是css样式

3. 属性绑定

<标签 style=”font-size:20px;border-color:red” />
